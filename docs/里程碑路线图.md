
### 🏁 Milestone 1: 基础设施与单体沙箱 (Weeks 1-3 | 2026年5月上中旬)
**阶段目标**：搭建好底层的物理环境，跑通单台机器上的网关管控与单一 Hermes 容器的工具调用。此时暂不引入多节点与复杂状态机。

* **P0: 控制平面基建 (FastAPI + MongoDB)**
    * 初始化 FastAPI 工程，集成 Pydantic 数据校验。
    * 设计并部署 MongoDB 的核心 Collection (`AgentNodes`, `ProjectWorkspaces`)。
    * 实现轻量级 API 网关，接管对外 LLM 请求并完成基础 Token 计费打点。
* **P0: 执行平面基建 (Docker Swarm)**
    * 编写标准化 `hermes-worker` Dockerfile。
    * 通过 Python `docker` 库（或直接调用 Swarm API）实现通过代码拉起、暂停和销毁隔离的 Agent 容器。
* **P1: 基础 MCP 协议与沙箱挂载**
    * 实现 Workspace Volume 挂载机制。
    * 开发第一批极简的 MCP Tools（`read_file`, `write_file`, `list_dir`），确保运行在容器内的 Hermes 能双向读写挂载的白板。

### 🏁 Milestone 2: 状态机流水线与扩展流 (Weeks 4-6 | 2026年5月下旬至6月初)
**阶段目标**：让系统具备“多智能体协作”的能力，引入流水线控制权交接、数据快照回滚以及基于 Python 装饰器的技能注册中心。

* **P0: 严格流水线与快照流转**
    * 开发控制平面的流水线引擎，实现独占式的沙箱访问权（文件锁机制）。
    * 实现基于 Docker Volume 的轻量级快照打包与恢复机制 (`tar.gz` 归档到 GridFS 或本地目录)。
* **P0: 逆流与强制纠错闭环**
    * 开发 `reject_to_previous_node` 驳回指令接口。
    * 实现底层拦截逻辑：回滚沙箱快照，并将审查 Agent 的驳回意见封装为 `[CRITICAL SYSTEM FEEDBACK]` 强行注入生成 Agent 的下一步上下文中。
* **P1: 技能注册中心 (Skill Registry) v1.0**
    * 实现 `@AgentSkill` 装饰器，支持将 Python 函数自动解析为 JSON Schema。
    * 完成基础的 Skillpack 本地目录结构设计与动态环境变量注入。

### 🏁 Milestone 3: Pre-Alpha Demo 交付阶段 (Weeks 7-8 | 2026年6月)
**阶段目标**：引入全局调度器与人工接管机制，彻底跑通 AI 驱动的动态编排。在 6 月底前，使用该原型成功跑通“故事工厂”的复杂世界观设定与逻辑审查场景。

* **P0: 全局调度器 (Global Coordinator)**
    * 赋予全局 Agent `spawn_worker_agent` 和 `create_issue` 的核心 MCP 权限。
    * 实现基于大模型意图分析的动态容器拉起（无需预先写死 YAML）。
* **P0: Human-in-the-Loop (时停与人工接管)**
    * 在网关层定义“关键事务拦截点”（如修改核心设定文件）。
    * 实现底层的逻辑挂起（Suspend）状态，等待接收人类指令后再执行状态注入（Approve/Reject）。
* **P1: Pre-Alpha 成果验收（业务联调）**
    * 导入故事工厂模板：全局 Agent 自动拉起架构师、主笔和审查员。
    * 跑通带有陷阱逻辑的系统规则更新场景，验证人工审批机制是否能完美截停并让管理员确认操作。

### 🏁 Milestone 4: 面向开源的控制台与生态 (2026年7月及以后)
**阶段目标**：将硬核的底层后端封装为具备极佳开发者体验（DX）的完整开源产品，并建设社区能力。

* **P1: “三位一体” Web Dashboard**
    * 基于 React Flow 渲染全局 Agent 动态生长的流水线拓扑图。
    * 集成 Monaco Editor 打造浏览器端的白板沙箱视图（实时 Diff 呈现）。
    * 打造全链路 Token 流与 MCP 调用的时间轴监控器。
* **P2: 生态与分发**
    * 发布官方 GitHub 仓库及完善的双语 README。
    * 提供基于 `docker-compose.yml` 的单机一键本地部署方案 (Local First)。
    * 建立远端 Skill Hub 索引，支持一键拉取社区优质技能包。
